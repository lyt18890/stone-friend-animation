# 三维动画短片《石头朋友》制作流程

## 🎬 项目简介
本短片使用AI生成3D模型，结合Blender手动K帧制作完成。故事改编自海绵宝宝片段，讲述派大星向海绵宝宝介绍他的“石头朋友”。

**时长**：1分钟  
**使用工具**：Mesh、Blender、Mixamo、剪映

---

## 🛠️ 制作流程

### 1. AI生成3D模型
使用**Mesh**生成三个角色模型：
- 海绵宝宝
- 派大星
- 菠萝屋
https://github.com/lyt18890/stone-friend-animation/blob/main/%E5%B1%8F%E5%B9%95%E6%88%AA%E5%9B%BE%202026-06-04%20001444.png
使用Mesh自动绑骨骼然后导出为 FBX格式，导入 Blender。

### 2. 场景搭建
- 创建沙滩地面（平面+材质）
- 添加天空背景（世界环境+纹理）
- 添加石头道具（3D模型导入）

https://github.com/lyt18890/stone-friend-animation/blob/main/%E5%B1%8F%E5%B9%95%E6%88%AA%E5%9B%BE%202026-05-14%20113525.png


### 3. 音频制作
- 爱给网上找到海绵宝宝和派大星的音频，UBERDUCK网站上生成角色配音
- 在剪映中剪辑音频，导出 MP3
- 导入 Blender 作为动画节奏参考
https://github.com/lyt18890/stone-friend-animation/blob/main/%E5%B1%8F%E5%B9%95%E6%88%AA%E5%9B%BE%202026-05-31%20182042.png
### 4. 角色动画
- Mixamo上下载动作
- 用不了动作的地方手动K帧
  
  下载动作
(https://github.com/lyt18890/stone-friend-animation/blob/main/%E5%B1%8F%E5%B9%95%E6%88%AA%E5%9B%BE%202026-06-11%20232638.png)



### 5. 输出与合成
- 由于没时间渲染了，我直接使用录屏方式录制Blender视口制作动画各片段
- 在剪映中剪辑动画素材，对齐音频
- 导出最终 MP4 文件
https://github.com/lyt18890/stone-friend-animation/blob/main/%E5%B1%8F%E5%B9%95%E6%88%AA%E5%9B%BE%202026-06-12%20011256.png

## ✅ 总结
通过本次项目，我了解了从AI生成模型到Blender手动K帧、再到最终输出的完整3D动画制作流程。虽然遇到了Mixamo绑定失败、动作库不匹配等问题，但通过手动K帧和录屏输出等方式灵活解决，最终完成了作品。
